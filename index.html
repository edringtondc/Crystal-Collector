Crystal Collection Game

Game Rules

You will be given a random goal number at the start of the game.

There are four crystals below. By clicking on a crystal you will add a specific value to your number of crystals collected.

You win the game by matching your total crystals collected to the goal number.

You lose the game if you collect more crystals than the goal.

The value of each crystal is hidden from you until you click on it.

Each time when the game re-starts, the game will change the values of each crystal.
